# Drum Application

This project is a web application that allows you to play various drum sounds. Users can trigger these sounds by either clicking on buttons or pressing specific keyboard keys.

## Features

- **Button-Based Sound Playback**: Click on buttons on the screen to play different drum sounds.
- **Keyboard Shortcuts**: Press specific keys on the keyboard to play corresponding drum sounds.
